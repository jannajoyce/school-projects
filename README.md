## 📌 Projects

### 🐍 SnakeAI: Snake AI-Based Classification and Anti-venom Locator System (2025)
- Built a public safety and wildlife conservation system using YOLOv8n and TensorFlow Lite (FP16)
- Integrated trained models into a mobile application for snake identification
- Enabled rapid classification and anti-venom location support for emergency response

---

### 🏭 Item Monitoring and Counting System for Conveyor-Based Production Lines (2025)
- Designed a real-time item counting system using ESP32 and FreeRTOS
- Implemented laser–LDR sensing to detect items and compute items-per-minute (IPM)
- Displayed live metrics (current IPM, total count, previous IPM) on an OLED screen
- Enabled operator reset via hardware button and ensured persistence using EEPROM
- Implemented serial input for expected IPM with visual alerts on performance deviation
- Utilized concurrent FreeRTOS tasks for sensing, display updates, logging, and alerts

---

### 💡 Smart Mood Lamp (2024)
- Developed an interactive lighting system with automatic color changes based on ambient temperature
- Implemented clap-based on/off control using interrupt-driven input handling

### 🤖 Wall Following Robot Using Bare-Metal Programming (2024)
- Implemented an autonomous wall-following robot entirely in C without an OS
- Replaced blocking delays with a 1 ms Timer0 CTC interrupt for precise timing
- Used interrupt-based ultrasonic echo capture for accurate distance measurement
- Implemented bang-bang steering control (PID-extendable) for real-time navigation
- Achieved deterministic and responsive control through ISR-driven architecture

- Verified functionality through simulation and instruction-level testing
